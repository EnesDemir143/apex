## 1. SQLAlchemy 2.0 Async Models

- Use `DeclarativeBase` with `Mapped[]` annotations
- `mapped_column()` replaces `Column()`
- Async engine: `create_async_engine("postgresql+asyncpg://...")`
- Vector type from pgvector: `from pgvector.sqlalchemy import Vector`
- Financial precision: `Numeric(18, 8)` for prices, `Numeric(24, 8)` for volume, `Numeric(5, 4)` for confidence
- UUID PK: `mapped_column(UUID, primary_key=True, server_default=text("gen_random_uuid()"))`

## 7. Embedding Model-Agnostic Design

- Vector dimension read from `settings.embedding_dimension` (default: 768)
- Embedding model specified in `settings.embedding_model` (default: "nomic-embed-text-v2-moe")
- Do not hardcode dimension in model — use config value
- Default Nomic Embed Text V2: 768-dim native, MRL supports truncation to 256
- **Context window: 512 tokens** (NOT 8K) — chunking must respect this
