<action>
Create base.py with DeclarativeBase. Create all model files using SQLAlchemy 2.0 Mapped[] syntax.

**Key tables and design rules:**

1. **stocks**: id, ticker (unique, String(10)), name, sector, exchange, is_active (bool default true), created_at, updated_at
2. **stock_prices**: id (BigInteger), stock_id (FK), date, open/high/low/close/adj_close as Numeric(18,8), volume as Numeric(24,8), source (String), ingested_at. UniqueConstraint on (stock_id, date).
3. **ingestion_log**: id, ticker, source, status, rows_ingested, rows_updated, started_at, completed_at, error_message
4. **analysis_runs**: id (UUID, server_default=gen_random_uuid()), stock_id (FK), final_signal, final_confidence Numeric(5,4), status, summary (JSONB), total_tokens, cost_usd, latency_ms, environment, compaction_applied (bool), created_at
5. **agent_decisions**: id, analysis_run_id (FK → analysis_runs.id), agent_name, prompt_version (int), signal, confidence Numeric(5,4), reasoning (JSONB), indicators (JSONB), tokens_input, tokens_output, cost_usd, latency_ms, is_fallback (bool), error_message, created_at
6. **embeddings**: id (BigInteger), stock_id (FK), content (Text), embedding (Vector — read dimension from config, default 768), content_type, metadata (JSONB), created_at
7. **trades**: id, stock_id (FK), analysis_run_id (FK → analysis_runs.id, nullable), signal, confidence Numeric(5,4), entry_price/exit_price/pnl Numeric(18,8), status (proposed/accepted/rejected), environment, notes (Text), created_at
8. **prediction_band_log**: id, stock_id (FK), analysis_run_id (FK → analysis_runs.id), predicted_for (Date), band_upper/band_lower/band_mid/actual_close Numeric(18,8), predicted_at
9. **llm_usage_log**: id (BigInteger), analysis_run_id (FK nullable), agent_name, model, tokens_input, tokens_output, cost_usd, latency_ms, cache_hit (bool), cache_key (String), created_at

**DO NOT** create agent_checkpoints table — LangGraph auto-creates via AsyncPostgresSaver.setup() in Phase 7.

Export all models from __init__.py.
</action>
