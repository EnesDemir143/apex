<read_first>
- src/apex/agents/state.py

</read_first>
<action>
Create each agent as async function taking AgentState, returning partial state update.
Technical: calculate indicators → LLM interprets → technical_analysis dict.
Fundamental: RAG stub retrieves context → LLM analyzes → fundamental_analysis dict.
Risk: calculate volatility, max_drawdown → LLM assesses → risk_assessment dict.
Portfolio Manager: reads all analyses → LLM synthesizes → portfolio_decision with signal and confidence.
All agents catch exceptions and append to errors list instead of raising.

**LangSmith Integration:**
- Each LLM call MUST pass `config={"run_name": "<agent_name>", "metadata": {"ticker": state["ticker"], "agent": "<agent_name>"}}` to `ainvoke()`
- This ensures every agent call is individually traced and filterable in LangSmith UI
- LANGCHAIN_TRACING_V2=true env var auto-enables tracing (set in Settings from Phase 1)
</action>
<acceptance_criteria>
- technical.py contains `async def technical_agent(state: AgentState)`
- fundamental.py contains `async def fundamental_agent(state: AgentState)`
- risk.py contains `async def risk_agent(state: AgentState)`
- portfolio_manager.py contains `async def portfolio_manager(state: AgentState)`
</acceptance_criteria>
</task>
