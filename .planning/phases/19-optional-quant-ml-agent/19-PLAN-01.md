<architecture>
## Ensemble Architecture

### Training Pipeline (Snakefile_train)
```
OHLCV data (yfinance fallback)
  → Feature engineering (price changes, returns, volatility, rolling stats, indicators)
  → Train/val split (time-series aware)
  → 4 models: RandomForest, XGBoost, LightGBM, CatBoost
  → Soft voting ensemble (weighted: RF=0.20, XGB=0.25, LGBM=0.25, Cat=0.30)
  → Evaluate + save with joblib → models/quant/
        ├── random_forest.pkl
        ├── xgboost.pkl
        ├── lightgbm.pkl
        ├── catboost.pkl
        ├── ensemble_voter.pkl
        ├── scaler.pkl
        └── metadata.json
```

### Inference (in Apex)
```
OHLCV → Features → Scale → 4 models predict → Ensemble vote
  → signal + confidence + top features + model_version + device
```

### Dependencies
- `scikit-learn` — RandomForest + scaler + voting
- `xgboost` — XGBClassifier
- `lightgbm` — LGBMClassifier
- `catboost` — CatBoostClassifier
- `joblib` — model save/load
- `snakemake` — training pipeline orchestration
</architecture>
